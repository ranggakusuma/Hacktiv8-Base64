# Hacktiv8-Base64
Helpers to encode or decode base64

## __Install__
```
npm install --save hacktiv8-base64
```
## __Usage__

```javascript
const base64 = require('hacktiv8-base64');
// or
import base64 from 'hacktiv8-base64';

// ...

base64.encode('I love base64');

base64.decode('SSBsb3ZlIGJhc2U2NA==');
```
